# 🛒 PrePrueba - Mini Ecommerce

## 📌 Descripción

Este proyecto es una aplicación web tipo ecommerce desarrollada con **JavaScript Vanilla**, donde existen diferentes roles de usuario como:

- Cliente
- Seller
- Admin

La aplicación permite visualizar productos y realizar operaciones CRUD usando una API simulada con **JSON Server**.

---

# 🚀 Tecnologías utilizadas

- HTML5
- Tailwind CSS
- JavaScript (ES6 Modules)
- JSON Server
- LocalStorage

---

# ⚙️ Funcionalidades

## 👤 Cliente

- Ver productos
- Visualizar stock y precio
- Interfaz de carrito de compras
- Logout

## 🛍️ Seller / Admin

- Crear productos
- Editar productos
- Eliminar productos
- Gestión de inventario
